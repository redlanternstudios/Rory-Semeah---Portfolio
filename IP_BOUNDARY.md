# IP Boundary

This repo is public by design.

It may contain:
- summaries
- screenshots
- high level architecture
- redacted case studies
- outcomes and lessons

It must not contain:
- proprietary source code
- private prompts
- database secrets
- exact workflow logic
- internal credentials
- anything that would let someone rebuild the edge

Rule:
If a detail helps an employer understand the value but also helps a competitor copy the mechanism, keep it out.
