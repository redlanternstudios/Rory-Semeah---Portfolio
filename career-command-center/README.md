# Career Command Center

Target role: Developer Success Engineer at Vercel

## Match score rule

1. Score every role out of 100 before drafting anything.
2. If the score is under 70, stop and ask bridging questions before writing the final resume or letter.
3. Loop until the gap is closed or the user decides to proceed with the honest score.
4. Every role specific draft must start with the score, the missing keywords, and the proof gaps.
5. Every tool or platform mention must include the capacity it was used in.

## Trigger loop

- job post arrives
- extract role title, duties, tools, and ATS phrases
- score against the master CV
- if score is 70 or higher, draft
- if score is below 70, ask only the questions that can raise the score
- rescore after answers and repeat

## Hook

This rule applies to:
- ATS resume
- company directed cover letter
- recruiter packet
- any future role specific version
